# People & Roles

Manage your organization's team structure with people, roles, and functions.

## People

The People section allows you to:
- **Add team members** to your organization
- **Assign roles** to define responsibilities
- **Track assignments** across functions

### Adding a Person

1. Navigate to **People** in the sidebar
2. Click **Add Person**
3. Fill in their details (name, email, etc.)
4. Assign appropriate roles

## Org Chart

Visualize your organization's hierarchy:
- See reporting relationships
- Add vacant positions
- Drag and drop to reorganize

### Creating the Org Chart

1. Go to **Org Chart** in the sidebar
2. Click **Add Position** to start
3. Add roles and assign people
4. Connect positions to show reporting lines

## Roles

Roles define sets of responsibilities:
- Group related functions together
- Assign permissions and access
- Create templates for common positions

### Creating a Role

1. Navigate to **Roles**
2. Click **New Role**
3. Add a name and description
4. Assign functions to the role

## Functions

Functions are atomic units of work:
- Define specific responsibilities
- Assign to roles or directly to people
- Track completion and status

### Creating a Function

1. Go to **Functions**
2. Click **Add Function**
3. Describe what the function involves
4. Assign it to relevant roles
